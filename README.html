The Layout Library, release 1.4.0

What is it?

The Layout Libary (liblayout) is an award winning shared library that allows programmers to quickly and easily create fully font-sensitive and/or resizable GUI's for their programs. liblayout also supports color-drops from applications such as roDesign's roColour and Beatware's Painter, and it supports font-drops from FontSelector or any other application that uses Dianne Hackborn's FFont class for storing fonts in BMessages.
