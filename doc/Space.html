The Layout Library: Space

Derived from: public MView, public BView

Declared in: <Space.h>

Overview

Space is a simple class that does nothing but take up space. Use it wherever you want some extra space between other classes.

Constructor and Destructor

Space()

    Space()
    Space(minimax)
    Space(BMessage*)
Initializes the Space with the given minimum and maximum size and weight, and sets its color to FILL_COLOR. If no constraints are specified, the Space will have a minimum size of zero, and an "infinite" maximum size.
